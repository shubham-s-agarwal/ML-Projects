Performing Analysis of Deep Learning Models for Q&A classification

The configuration files for 2 models: BiLSTM and BOW contains editable parameters for the models. 

1) To execute the python code, do the following:
	a) Open terminal and navigate to the src folder.
	b) Execute the python file using the following command:

		> python3 question_classifier.py --train --config [configuration_file_path]

2) To change the configurations, navigate to the Data Folder, and make changes in the corresponding config file for the required model.

Libraries to be installed:

1) Python 3.6+ 
2) PyTorch
